# World Imaginer

Research repository for procedural terrain generation, inspired by games like Minecraft and Terraria.

Each sub-folder is an independent project exploring a different aspect of procedural generation.

## Projects

| # | Project | Description |
|---|---------|-------------|
| 01 | [Noise Fundamentals](./01-noise-fundamentals) | Perlin & Simplex noise implementation and visualization |
| 02 | [2D Terrain](./02-2d-terrain) | Side-scrolling terrain with caves, biomes, and water |
| 03 | [Tilemap World](./03-tilemap-world) | Infinite chunk-based world with ores, trees, and LRU caching |
| 04 | [3D Voxel Terrain](./04-3d-voxel-terrain) | Minecraft-style 3D world with Three.js and greedy meshing |

## Stack

- TypeScript
- Vite
- Canvas API / Three.js
